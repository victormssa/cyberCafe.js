Função 

em JavaScript

 Função (Function), é um procedimento dentro do JavaScript, que por si é um conjunto de instruções para executar uma tarefa. Podendo possuir parâmetros para ser inicializado, ou não.

Exemplo

                    
                    function somar() {
                        var tn1 = window.document.getElementById("txtn1");
                        var tn2 = window.document.querySelector("input#txtn2");
                        var res = window.document.getElementById("res");
                        var n1 = Number(tn1.value);
                        var n2 = Number(tn2.value);
                        var s = n1 + n2;
                        res.innerHTML = `A soma entre ${n1} e ${n2} é equivalente a ${s}.`;
                        }

 Olhando pro exemplo de uma função acima podemos determinar que ele não possui um parâmetro próprio para se inicializar, logo precisa ser declarado dentro de um código.

ID 

ou 

Class 

 Em referente a estilização e a funcionalidade de nosso script dentro de uma página da nossa aplicação, é uma facilidade poder nomear/diferenciar certos elementos entre eles, para isso temos dois Atributos que nos auxiliam: O Id e o Class.

 O atributo Id se distingue por ser uma identificação única para os elementos em HTML. Enquanto o atributo Class se distingue por ser reutilizado mais de uma vez dentro do script. No CSS, O atributo Class é referenciado pelo ponto(.), enquanto o Id pela cerquilha(#).

getElementById 

e 

getElementByClassName 

 getElementById e getElementByClassName são dois metódos utilizados no JavaScript para capturar elementos. Em suas especificações o getElementById captura somente um elemento enquanto o getElementByClassName cria um Array com todos os elementos da Classe(Class) requisitada.

querySelector 

em JavaScript

 O querySelector é um metódo por si mais recomendável e atual, pois trabalha com ambos Id e Class, mas retorna somente o primeiro elemento buscado dentro da aplicação.
