Função 

em JavaScript

    Função (Function), é um procedimento dentro do JavaScript, que
    por si é um conjunto de instruções para executar uma tarefa.
    Podendo possuir parâmetros para ser inicializado, ou não.

Exemplo


                
 function somar() {
    var tn1 = window.document.getElementById("txtn1");
    var tn2 = window.document.querySelector("input#txtn2");
    var res = window.document.getElementById("res");
    var n1 = Number(tn1.value);
    var n2 = Number(tn2.value);
    var s = n1 + n2;
    res.innerHTML = `A soma entre ${n1} e ${n2} é equivalente a ${s}.`; 
    }
                
            
 
                Olhando pro exemplo de uma função acima podemos determinar que 
                ele não possui um parâmetro próprio para se inicializar, logo precisa
                    ser declarado dentro de um código.
